# 26W_CST8914 Accessible by Design
# Final Project: Empower Ability Labs SPA- Group 2

## Live Site
- https://shan-ac.github.io/CST8914-Final-Project/​

## Project Overview
This project is an accessible Single-Page Application (SPA) created for the CST8914 final project. It was built using **HTML, CSS, and JavaScript**.

The website includes:
- Home
- Services
- Schedule a Call

## Main Features
- Single-page navigation without full page reload
- Dynamic page titles
- Focus management after navigation
- Accessible modal dialog
- Accessible switch control
- Show/hide event details field
- Accessible form validation and user notifications
- Responsive layout for desktop and mobile

## Accessibility
This project was designed to follow **WCAG 2.1 AA** and relevant **ARIA Authoring Practices Guide (APG)** patterns.

Accessibility features include:
- Semantic HTML structure
- Keyboard-accessible navigation and controls
- Visible focus indicators
- ARIA attributes for interactive components
- Error messages connected to form fields
- Live region announcements for form feedback

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap CSS only

## Project Structure
- `EmpowerAbilityLab.html` - main page structure
- `EmpowerAbilityLab.css` - custom styling and responsive design
